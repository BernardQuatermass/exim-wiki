Q5028
=====

Question
--------

When I HUP the Exim daemon, the name shown in the process table changes
from */usr/lib/sendmail* (which is a symlink) to the real binary name.
Can I change this?

Answer
------

Add this to your Exim configuration:

    exim_path = /usr/lib/sendmail
