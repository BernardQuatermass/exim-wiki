Q0024
=====

Question
--------

When I run `exim -bd -q10m` I get *PANIC LOG: exec of exim -q failed*.

Answer
------

This probably means that Exim doesn't know its own path so it can't
re-exec itself to do the first queue run. Check the output of

    exim -bP exim_path
