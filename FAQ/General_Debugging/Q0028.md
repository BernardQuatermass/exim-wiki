Q0028
=====

Question
--------

Exim is reporting IP addresses as 0.0.0.0 or 255.255.255.255 instead of
their correct values. What's going on?

Answer
------

You are using a version of Exim built with gcc on an IRIX box. See
\`FAQ/IRIX/Q9502\`\_.
